QUESTION 1.1.1
Show $\{E_y\}$ is a valid POVM.

ANSWER
1) Sum is the identity \[ \sum_y{E_y} = \sum_y{\ket{e_y}\bra{e_y}} = \sum_y{\rho^{-1/2}\ket{\tilde{\Psi_y}}\bra{\tilde{\Psi_y}}\rho^{-1/2}} = \rho^{-1/2}(\sum_y{\ket{\tilde{\Psi_y}}\bra{\tilde{\Psi_y}}})\rho^{-1/2} = ... \] Applying $\ket{\tilde{\Psi_y}} = \sqrt{p_y}\ket{\Psi_y}$ where $p_y > 0$ \[ ... = \rho^{-1/2}(\sum_y{\ket{\tilde{\Psi_y}}\bra{\tilde{\Psi_y}}})\rho^{-1/2} = \rho^{-1/2}(\sum_y{\sqrt{p_y}\ket{\Psi_y}\bra{\Psi_y}\sqrt{p_y}})\rho^{-1/2} = \rho^{-1/2}(\sum_y{{p_y}\ket{\Psi_y}\bra{\Psi_y}})\rho^{-1/2} = ... \] Applying the definition $\rho = \sum_y{p_y\ket{\Psi_y}\bra{\Psi_y}}$ we get finally \[ ... = \rho^{-1/2}(\sum_y{{p_y}\ket{\Psi_y}\bra{\Psi_y}})\rho^{-1/2} = \rho^{-1/2}\rho\rho^{-1/2} = \mathbb{1} \] 2) Positive Operator??
Let's expand $E_y$: \[ E_y = \ket{e_y}\bra{e_y} = \rho^{-1/2}\ket{\tilde{\Psi_y}}\bra{\tilde{\Psi_y}}\rho^{-1/2} = \rho^{-1/2}\sqrt{p_y}\ket{\Psi_y}\bra{\Psi_y}\rho^{-1/2}\sqrt{p_y} = \rho^{-1}p_y\ket{\Psi_y}\bra{\Psi_y} \] Self-adjoint. Using the fact the $\rho$ is positive and $\sqrt{p_y}$ is a real number \[ E_y^{\dagger} = (\rho^{-1}p_y\ket{\Psi_y}\bra{\Psi_y})^{\dagger} = \rho^{-1}p_y\ket{\Psi_y}\bra{\Psi_y} = E_y \]

QUESTION 1.1.2

Here trick, as always, is how to introduce the indentity and from the previous one we have a good candidate that uses $\rho^{-1/2}$ \[ \mathbb{1} = \sum_{k}E_{k} = \sum_{k}\ket{e_{k}}\bra{e_{k}} = \sum_{k}\ket{e_{k}}\bra{e_{k}} = \sum_{k}{\rho^{-1/2}\ket{\tilde{\Psi_k}}\bra{\tilde{\Psi_k}}\rho^{-1/2}} \] So applying that th $G_{xy}$ we get \[ G_{xy}= \bra{\tilde{\Psi_x}} \ket{\tilde{\Psi_y}} = \bra{\tilde{\Psi_x}} \mathbb{1} \ket{\tilde{\Psi_y}} = \bra{\tilde{\Psi_x}} (\sum_{k}{\rho^{-1/2}\ket{\tilde{\Psi_k}}\bra{\tilde{\Psi_k}}\rho^{-1/2}}) \ket{\tilde{\Psi_y}} = \sum_{k} { \bra{\tilde{\Psi_x}} \rho^{-1/2} \ket{\tilde{\Psi_k}} \bra{\tilde{\Psi_k}} \rho^{-1/2} \ket{\tilde{\Psi_y}} } \equiv \sum_{k} { S_{xk}S_{ky} } \] where \[ S_{xy} \equiv \bra{\tilde{\Psi_x}} \rho^{-1/2} \ket{\tilde{\Psi_y}} \]
